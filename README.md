# webdriver-repl

`REPL` interface to learn and debug the selenium webdriver API from terminal

# Prerequisite

Requires JDK > 8 to be installed.

# Installation

```shell
npm install -g webdriver-repl
```

system.

# Usage

```shell
wd-repl --browser chrome
```
